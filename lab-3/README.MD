Запуск под Windows
---
+ lua 5.4.2 или новее
+ lua54 в Path
+ \> run.bat

Примечание
---
Баги можно искать штатными средствами, изменив параметр CFG_CHECK_EQUAL на truе.
Так, после преобразований, грамматики будут проверяться на эквивалентность исходной (на не очень длинных словах).
